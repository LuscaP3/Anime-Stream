# 🎌 AnimeStream

Um site desenvolvido com React + Vite para explorar o universo dos animes!  
Aqui você poderá descobrir, pesquisar e acompanhar seus animes favoritos de forma simples e rápida.

---

## 🚧 Status do Projeto

> ⚠️ **Este projeto ainda está em desenvolvimento!**

Algumas funcionalidades estão incompletas ou sofrer podem alterações.

---

## ✨ Funcionalidades (em progresso)

- 🔍 Busca de animes
- 📄 Página de detalhes
- 📊 Informações como nota, sinopse e gênero
- 📱 Layout responsivo
